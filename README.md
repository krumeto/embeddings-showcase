# embeddings-showcase
A repo to showcase what one can do with dense embeddings. Embeddings are cheap, flexible and intuitive, and a very helpful tool to utilize.

## Areas to showcase:

1. Simple vector search
2. Clustering
3. Topic modelling
4. Classification

## Dataset to use - a subset of https://huggingface.co/datasets/dell-research-harvard/newswire - see the example

## Streamlit for interface

## Repo directions:

1. Stay readible and explainable.
2. Minimize dependencies. For example, it is not needed to create a full blown faiss index for a small dataset. Utilize available tools - numpy, parquet, etc..
3. Prioritize learning and interactivity, vs. production-level code.
